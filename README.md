# ♻️ ELV 廢塑膠產銷履歷示範平台

## 🎯 專案目標
建立廢塑膠（來源自報廢車輛）從出廠到產品銷售的完整履歷追蹤系統，透過QR碼掃描登錄提升可追溯性，作為未來歐盟ELV廢塑膠再利用政策接軌的示範平台。

## 🔄 作業流程
1. **產生QR碼** - 稽核團體為每批廢塑膠產生唯一QR碼
2. **階段登錄** - 各節點人員（出廠→運輸→處理→再製→銷售）掃描QR碼登錄資料
3. **履歷追蹤** - 完整記錄時間、重量、處理地點、產品型號等資訊
4. **資料管理** - 支援查詢、匯出，資料保存至少五年
